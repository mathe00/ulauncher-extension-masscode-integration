Install the required Python dependencies by running this command
from your terminal (adjust the path to match YOUR extension directory):

    pip install -r requirements.txt -t libs/

To find your extension directory, run:

    ls ~/.local/share/ulauncher/extensions/com.github.mathe00.*

Then cd into that directory and run the pip install command above.

Example (your directory name may differ):
    cd ~/.local/share/ulauncher/extensions/com.github.mathe00.ulauncher-extension-masscode-integration/
    pip install -r requirements.txt -t libs/
